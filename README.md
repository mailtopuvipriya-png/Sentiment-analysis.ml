# Sentiment Analysis ML Project

## Description
This project predicts sentiment of reviews using Machine Learning.

## Dataset
Amazon Reviews Dataset (Kaggle)

## Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF
- LinearSVC

## How it works
1. Load dataset
2. Clean text
3. Train ML model
4. Predict sentiment

## Output
- Accuracy: ~80%
- Classes: Positive / Negative
